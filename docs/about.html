Project History

In December of 2016, Students from the University of East Anglia located in Norwich, England took on a Study Planner project as part of their course work for "CMP-5012B - Software Engineering I". This led to the creation of the "PearPlanner", which our current project's code is built off using Git-Hub as the file storage repository. The purpose of this project was to load a file that contains the pertinent information from a given semester (coursework, deadline information, exam information etc.) with a simple Graphical User Interface. If you want more information on the original project (PearPlanner), click HERE to download the PDF. Their four central capabilities are required by the solution are listed below

  1. Load module, coursework and deadline information from a defined file format
  2. Ability to define study tasks, details, milestones and deadlines
  3. Ability to record study activities that contribute towards completing study tasks and milestones
  4. Visualise(sic) activities, dependencies, intermediate milestones and deadlines in a Gantt chart representation as well as a study progress dashboard that highlights upcoming deadlines, progress towards completing milestones and time spent for each module

Credit for the original project goes to: Benjamin Dickson, Andrew Odintsov, Zilvinas Ceikauskas and Bijan Ghasemi Afshar

Key Features

The current implementation of the RaiderPlanner, formerly the PearPlanner has the 4 central capabilities from the original project still at its core, with the implementation of several changes made by the CEG-3120 students. This planner is one that can be used by a student to keep track of the major items due over the course of the semester. Tools that allow the user to add calendar reminders, make notes and assign tasks for each assignment due.
